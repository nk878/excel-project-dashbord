# Vedent Store Sales Analysis (Excel Project)

## Project Overview
This project analyzes **Vedent Store sales data** using **Microsoft Excel** to understand sales performance, customer behavior, and product trends.  
It is created as a **portfolio project** for Data Analyst / Business Analyst roles.

## Tool Used
- Microsoft Excel

## Dataset Description
The dataset contains sales transactions from Vedent Store, including:
- Order ID
- Order Date
- Customer Name
- Gender
- Age
- Product Category
- Quantity
- Sales Amount
- City
- State

## Excel Work Performed
- Data cleaning and formatting
- Removing duplicates and handling missing values
- Creating calculated columns (Month, Year, Age Group)
- Using Excel formulas:
  - SUM, AVERAGE, COUNT
  - IF, VLOOKUP / XLOOKUP
- Creating Pivot Tables for analysis

## Analysis & Dashboard
- Monthly and yearly sales trend
- Category-wise sales performance
- Gender-wise revenue distribution
- Top cities and states by sales
- Interactive dashboard using:
  - Pivot Charts
  - Slicers
  - Timelines

## Key Insights
- Certain product categories contribute the highest revenue
- Sales show clear monthly trends
- Metro cities generate higher sales compared to other regions
